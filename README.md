### Description

Conversion of temperature from one unit of measurement to another. After running the program, enter the value, source unit, and end.

### Launch program

Use command:

> ruby lib/main.rb

### Launch rubocop:

Use command:

> rubocop -a

### Launch tests:

Use command:

> rspec
